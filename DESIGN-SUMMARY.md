# Design summary

- Design concept: Overseas coverage board
- Layout personality: global editorial
- Typography: Spectral headings with Inter body text
- Color palette: primary #1e1b4b, secondary #a5b4fc, accent #f59e0b, background #eef2ff, text #1e1b4b, muted #64748b
- Shape language: Mixed geometric shapes
- Visual motif: time-zone ribbons and world bands
- Unique layout choices: first sections use Coverage atlas, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
